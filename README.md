#### Dev requirements
- Node.js or Bun

#### Dev
`npm install`

`npm run dev`

or

`bun install`

`bun --bun run dev`

#### Building for KA
Run

`node build.js` or `bun build.js`

This script puts all javascript and CSS into a single `output.html` file. This file should be able to be copied directly into the KA editor or run standalone anywhere.
